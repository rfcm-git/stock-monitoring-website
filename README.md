# 📈 Stock Monitoring Website

A web application that allows users to monitor real-time stock prices, track market trends, and manage a personalized watchlist. This project is designed to make stock tracking simple, fast, and accessible.

---

## 🚀 Features

- 📊 Real-time stock price updates  
- ⭐ Add/remove stocks from a watchlist  
- 🔍 Search stocks by ticker or company name  
- 📉 View historical price charts  
- 📱 Responsive design for mobile and desktop  
- ⚡ Fast and lightweight interface  

---

## 🛠️ Tech Stack

- Frontend: HTML, CSS, JavaScript / React (update if different)  
- Backend: Node.js / Express / Django (update if different)  
- API: Stock market data API (e.g., Alpha Vantage, Finnhub, etc.)  
- Database: MongoDB / MySQL / Firebase (if used)

## ⚙️ Installation

### 1. Clone the repository

git clone https://github.com/rfcm-git/stock-monitoring-website.git


### 2. Go to project folder

cd stock-monitoring-website


### 3. Install dependencies

npm install


### 4. Run the project

npm start


Then open:

http://localhost:3000


---

## 🔑 Environment Variables

Create a `.env` file in the root directory:


API_KEY=your_api_key_here
BASE_URL=your_api_base_url_here


---

## 📊 How It Works

- User searches for a stock symbol  
- App fetches live market data from API  
- Data is displayed in charts and tables  
- Users can save stocks to a watchlist
